PROYECTO 2.0 — BIOASISTENCIA ESTRUCTURADO CON BASE OFICIAL

CARPETAS

api/        Comunicación con Arduino y estado biométrico
assets/     CSS, JavaScript, logo e imágenes
config/     Configuración general, rutas y conexión PDO
database/   Backup/referencia de la base de datos
docker/     Archivos Docker (init.sql, data.sql, apache-vhost.conf)
exports/    Generación de reportes Excel
includes/   Sesiones, funciones compartidas y menú
pages/      Pantallas del sistema

ARCHIVOS RAÍZ

Dockerfile          Imagen Docker del servidor web
docker-compose.yml  Orquestación de servicios (web + db)
Makefile            Comandos de construcción y gestión Docker
.env.example        Plantilla de variables de entorno

BASE DE DATOS

Esquema (para Docker):
  docker/init.sql      — Tablas, vistas, índices, config base
  docker/data.sql      — Datos de prueba (INSERTs)

Backup/referencia:
  database/bioasistencia.sql  — Copia completa del esquema + datos

EJECUCIÓN CON XAMPP

1. Copiar la carpeta dentro de C:\xampp\htdocs\
2. Encender Apache y MySQL
3. Abrir phpMyAdmin
4. Importar database/bioasistencia.sql
5. Revisar config/conexion.php
6. Abrir el proyecto desde localhost

EJECUCIÓN CON DOCKER

Opción 1 — Comando directo:
  docker compose up -d --build

Opción 2 — Con Make (recomendado):
  make up-build

Dirección:
  http://localhost:8080

COMANDOS MAKE

  make help         Mostrar todos los comandos disponibles
  make build        Construir imagenes Docker
  make up           Levantar servicios (detached)
  make up-build     Reconstruir y levantar servicios
  make down         Detener y eliminar contenedores
  make stop         Detener contenedores sin eliminar
  make restart      Reiniciar servicios
  make logs         Ver logs en tiempo real
  make logs-db      Ver solo logs de la BD
  make logs-web     Ver solo logs del web
  make status       Estado de los contenedores
  make shell-db     Shell MySQL en el contenedor
  make shell-web    Shell bash en el contenedor web
  make clean        Eliminar contenedores, redes y volumenes
  make clean-all    Eliminar todo incluyendo imagenes
  make reset        Limpiar y reconstruir desde cero
  make db-export    Exportar base de datos a archivo .sql
  make test         Verificar configuracion docker-compose
  make env          Mostrar variables de entorno
